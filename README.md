# Super-Pawn-AI-chess
An asymmetrical, AI-driven chess variant built from scratch in Python using Pygame. This project features standard chess rules alongside a custom game mode introducing a powerful new piece: the Super Pawn. The game includes a fully functional opponent driven by the Minimax algorithm with Alpha-Beta pruning.
